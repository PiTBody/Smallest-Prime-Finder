# Smallest-Prime-Finder
Finds smallest primes that end with a particular string of numbers.

To use, download SmallestPrimeFinder.java and FastSmallestCheker.java into the same folder and run SmallestPrimeFinder.java.

Currently the end and the desired amount of numbers checked need to be changed manually in-code. Might be changed later on.

The current end string is set to be a string of 11111 ones. The current amount of numbers to search through is set to be 100000.

The output .txt file contains all the strings of number below a certain threshold that, when concatenated with the end string in code will produce a prime.
